1) What is Python? What are the way to install python in your pc?
Ans : Python is a general-purpose coding language which is one of the most popular languages due to its simplicity. It is easy to understand & learn.

      Ways to install Python :
      - Using Anaconda and use Jupyter Notebook to run python
      - Using Anaconda and use spyder to run python
      - Download directly from Python.org and use it in Text editors etc
  
2)  What are code enviroments? Types?
    - A place where you can write, compile, and see the output of your code.
      Its type:
      - Text Editors eg: Atom, Visual Studio Code
      - Full IDEs eg: Spyder, Pycharm
      - Notebook Environments eg: Jupyter notebook
